# Commit Rules
✨ **Feat | 새로운 기능 추가**

📚 **Docs | md, text 파일 변경**

🎨 **Style | 코드 스타일 변경 (주석, 포맷팅)**

♻️ **Refactor | 리팩토링, 코드 간략화, 기능 수정**

✅ **Test | 테스트 모듈 추가**

🔧 **Chore | 경로 수정, 추가, 삭제**

🛠️ **Build | NPM 종속성 관련**

🗑️ **DEL | 파일, 디렉토리 삭제 관련**

📝 **Layout | 마크업 관련 커밋**
